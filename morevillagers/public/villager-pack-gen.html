Villager Pack Generator

Trades List Villager Pack Generator
Create custom villager professions, types, and trades for Minecraft without writing any code! You'll need the Villager API mod for Villagerpacks to work.

Pack Metadata

Pack Format: 48 (Minecraft 1.21.1)
Namespace: Use lowercase letters, numbers, and underscores only
Note: Individual components default to "villagerapi" namespace if not specified

Workstation Blocks

Create custom workstation blocks for your villager professions. Upload textures for each face of the block (front, back, left, right, top, bottom). Leave namespace empty to use "villagerapi" namespace.

POI Types

POI (Point of Interest) types define which block a villager uses as their workstation. Reference your custom workstation blocks using the format "namespace:block_name" (e.g., "villagerapi:purpur_altar"). Leave namespace empty to use "villagerapi" namespace.

Villager Types

Villager types control the appearance based on biome. Upload a texture for each type. Leave namespace empty to use "villagerapi" namespace.

Professions

Professions define the villager's job, linking them to a POI type and work sound. Leave namespace empty to use "villagerapi" namespace.

Structure Tags & Map Decorations

Create structure tags for treasure maps and define custom map decoration icons. Upload a texture for the map decoration icon and choose the color for the map marker. Leave namespace empty to use "villagerapi" namespace.

Trades

Important: Trades must be valid JSON. Use the Validate button to check before generating.
Define what items villagers buy and sell at each level (1-5). You can reference custom structure tags in treasure map trades using the format "namespace:tag_name".

Biome-Specific Trades

Add extra/replaceable trades for villagers based on their biome/type.

Hero Gifts

Define what loot table is used when villagers throw gifts after winning a raid.

Biome Mappings

Control which villager type appears in which biome.

Loot Tables

Create loot tables for hero gifts. These will be placed in data/[namespace]/loot_table/gameplay/hero_of_the_village/
